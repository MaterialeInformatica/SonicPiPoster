# SonicPiPoster
This repo contains the material for the SIGCSE-TS-2024 poster Exploring Student Misconceptions about Concurrency Using the Domain-Specific Programing Language Sonic Pi by Daniele Traversaro, Giorgio Delzanno, Giovanna Guerrini.
Specifically it contains, in addition to the poster paper and the displayed poster, Chapter 3 of the PhD thesis by Daniele Traversaro describing the approach and including the relevant educational material (RAT questions and tAPP tasks). The folder Dataset contains the experimental results.
